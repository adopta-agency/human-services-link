The is a demo site for the Open Referral Human Services Data API (HSDA). The portal is fully equipped with all of the active API paths for v1.1 of the API specification allowing anyone to pull organizations, locations, and services from the demo implementation.

There is documentation, and code samples to get started, and the blog, roadmap, and other tools for staying in tune with what is happening with the platforms as we make changes.

The data is all Human Services Data Specification (HSDS) compliant and is the latest API implementation that follows the specification, helping make the human services more accessible and usable.

Please visit the support page, and stay in tune with the project using it's github repository. This is a community project so please get involved and let us know what you think.
